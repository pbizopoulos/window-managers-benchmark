X Window Managers Benchmark

This repository presents a benchmark for X window managers and for each one the following actions are automatically performed monthly:

  1. git clone repo
  2. sudo apt-get install benchmark-dependencies
  3. sudo apt-get install repository-dependencies
  4. premake (required actions before calling make)
  5. make

During the previous procedure multiple metrics are logged using the following commands:

  • first commit date: git log --format=%cs tail -1
  • last commit date: git log --format=%cs head -1
  • source lines of code (sloc): cloc .
  • make time: perf stat -r 10 -d sudo make -s -C wm clean install
  • binary size: stat -c %s wm
  • USS,PSS,RSS: xvfb-run -a wm & sleep 10 && smem -c "command uss pss rss"
  • VSZ: ps -o command,vsz
